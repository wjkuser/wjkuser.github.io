# [0763 划分字母区间](https://leetcode.cn/problems/partition-labels/description/)

> 中等 贪心

## 算法思路

根据题述，不难想到，如果我们知道每个字母最早出现的索引和最后出现的索引。那么可以将每个字母想象成一个首尾处于特定位置的水平线段，而划分依据则是找到尽可能多的垂直线，这些垂直线不会将水平线段切成两段。

因此，我最直观的想法是：将上述水平线段按照起始位置排序，并按顺序遍历，当下一线段的起止位置位于区间内，则加入并更新区间右界，否则形成一个划分。
由上述思想，此算法时间复杂度为 $O(n + | \varepsilon | \log | \varepsilon |)$，其中 $\varepsilon \leq 26$，即时间复杂度为 $O(n)$。空间复杂度为 $O(|\varepsilon |)$。

## 代码实现

```c title="partition-labels.c"
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
#define max(a, b) ((b) > (a) ? (b) : (a))
#define NUM 30

int a[NUM][2];

int cmp(const void* x, const void* y) {
    int xx = *(int*)x;
    int yy = *(int*)y;
    return a[xx][0] - a[yy][0];
}

int* partitionLabels(char* s, int* returnSize) {
    int n[NUM];
    memset(a, 0, NUM * 2 * sizeof(int));
    for (int i = 0; i < NUM; i++) {
        n[i] = i;
        a[i][0] = -1;
    }
    for (int i = 0; s[i]; i++) {
        if (a[s[i] - 'a'][0] < 0) {
            a[s[i] - 'a'][0] = i;
        }
        a[s[i] - 'a'][1] = i;
    }
    qsort(n, NUM, sizeof(n[0]), cmp);
    int index = 0, left = 0, right = a[n[0]][1], num = 0;
    int* ret = (int*)malloc(26 * sizeof(int));
    returnSize[0] = 0;
    while (index < NUM) {
        if (a[n[index]][0] < 0) {
            index++;
            continue;
        }
        if (a[n[index]][0] <= right) {
            right = max(right, a[n[index]][1]);
            index++;
        } else {
            returnSize[0]++;
            ret[num++] = right - left + 1;
            left = right + 1;
            right = a[n[index]][1];
        }
    }
    returnSize[0]++;
    ret[num++] = right - left + 1;
    return ret;
}
```

## UPDATE

阅读题解发现，可用贪心想法快速解决问题，不必对每个字母所在区间进行排序，而是在遍历中直接看当前字母出现的最后位置并不断更新当前划分的右界。当相等时，就意味着当前字母为当前划分的最后一个字符。
代码实现如下：

```c title="partition-labels.c"
int* partitionLabels(char* s, int* returnSize) {
    int last[26];
    int length = strlen(s);
    for (int i = 0; i < length; i++) {
        last[s[i] - 'a'] = i;
    }
    int* partition = malloc(sizeof(int) * length);
    int start = 0, end = 0;
    *returnSize = 0;
    for (int i = 0; i < length; i++) {
        end = fmax(end, last[s[i] - 'a']);
        if (i == end) {
            partition[(*returnSize)++] = end - start + 1;
            start = end + 1;
        }
    }
    return partition;
}
```
